# What is Ariawase?

Ariawase is an open source VBA library.

## Quick Start

Run `build.bat`, you'll get office macro-enabled file in bin directory.

### example

```ps
cscript vbac.wsf combine /src src2
cscript vbac.wsf run /target .\bin\ex007.xlsm
```
